Upgrade Acunetix Production Server to the Latest Supported Version


Upgrade the existing Acunetix Production server to the latest supported version following successful UAT validation. The activity includes pre-upgrade backup, software upgrade, post-upgrade validation, and verification of vulnerability scanning functionality.



Upgrade is required to maintain vendor support, security updates, bug fixes, and product stability. The previous version was outdated, reducing confidence in vulnerability assessment results. The upgrade was requested to ensure the platform continues to provide reliable and supported vulnerability scanning services.
